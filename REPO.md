
## Trabajo en el repositorio


### Entorno virtual Python y dependencias

**Activar entorno virtual Python**

En Linux:

~~~bash
source venv/bin/activate
~~~

En Windows:

~~~powershell
venv\bin\activate
~~~

**Instalar dependencias**

~~~
pip install -r requirements.txt
~~~

### Base de datos

Optamos por SQLite para tener una base de datos en un archivo y poder versionarlo.

Puede accederse con DBeaver o cualquier cliente de base de datos que pueda conectar a SQLite.
